== Descrição ==
**Habilitar Widgets Clássicos**
★★★★★<br>

> esc_attr(Restaurar e habilitar as telas de configurações de widgets clássicos anteriores do WordPress e desativar o editor de blocos Gutenberg para gerenciar widgets. Sem data de expiração!

O WordPress 5.8 transforma a tela de widgets clássicos em uma tela de widgets baseada em blocos. Sim, agora os widgets são blocos e isso estará disponível para todos instantaneamente.
Isso criará problemas de compatibilidade para muitos sites.

Há casos em que nosso plugin será necessário:
    * Temas quebrados ou saída peculiar.
    * Usuários simplesmente preferindo o sistema antigo.

Nosso plugin pode restaurar a tela de widgets clássicos com apenas um clique.
Não há outra configuração, as telas de configurações de widgets clássicos são habilitadas ou desabilitadas apenas ativando ou desativando este plugin.

Essa funcionalidade (restaurar widgets clássicos) também está disponível em nosso plugin gratuito WP Tools com outros 35 recursos úteis:
<a href="https://wordpress.org/plugins/wptools/">Plugin Grátis WP Tools</a> 

== Importante ==
Se você começar a usar o novo sistema de widgets baseados em blocos, perderá todos os seus blocos de widget ao ativar nosso plugin. 
Não há como voltar atrás, portanto, certifique-se de que é isso que você deseja. Os blocos antigos não reaparecerão se você mudar de ideia e desativar nosso plugin.

**Privacidade**

Este plugin não coleta ou armazena dados do usuário. Ele não define cookies e não se conecta a nenhum local de terceiros. Portanto, este plugin não afeta a privacidade do usuário de forma alguma.




== Instalação ==

1) Instale através do wordpress.org

2) Ative o plugin através do menu 'Plugins' no WordPress

ou

Extraia o arquivo zip e coloque o conteúdo no diretório wp-content/plugins/ de sua instalação do WordPress e, em seguida, ative o Plugin na página de Plugins.


**Desinstalação**

Para desinstalar/remover o plugin, visite a tela de Plugins, desative e exclua o plugin. Este plugin não faz alterações no banco de dados do WordPress.



== Perguntas Frequentes ==

**Existe um "Desfazer" para reverter a desativação?**

Sim, basta desativar o plugin. Quando o plugin não está ativo, o WordPress voltará ao comportamento padrão de widgets.
Se você começar a usar o novo sistema de widgets baseados em blocos, perderá todos os seus blocos de widget ao ativar nosso plugin. 
Não há como voltar atrás, portanto, certifique-se de que é isso que você deseja. Os blocos antigos não reaparecerão se você mudar de ideia e desativar nosso plugin.
